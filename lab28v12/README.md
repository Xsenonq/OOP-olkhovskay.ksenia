#  Лабораторна робота №28  
## Серіалізація об’єктів у JSON
##  Мета роботи
Навчитися серіалізувати та десеріалізувати складні об’єкти у формат JSON,  
зберігати дані у файли та завантажувати їх з використанням бібліотеки `System.Text.Json`.
##  Варіант 12: Ресторан
Реалізовано  класи предметної області:
- Dish — страва  
- Category — категорія  
- MenuItem — елемент меню  
- Menu — меню  
- MenuRepository — репозиторій  
###  Репозиторій (MenuRepository)
Реалізовані методи:

- Add() — додавання елемента  
- GetAll() — отримання всіх елементів  
- GetById(int id) — пошук за ID  
- SaveToFileAsync(string filename) — асинхронне збереження у JSON  
- LoadFromFileAsync(string filename) — асинхронне завантаження з JSON
