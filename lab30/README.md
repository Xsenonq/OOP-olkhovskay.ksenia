# Лабораторна робота №30  
## Тема: Написання юніт-тестів з xUnit

## Мета 
Навчитися створювати та запускати юніт-тести для бізнес-логіки за допомогою **xUnit**, використовуючи різні типи assertion та параметризовані тести.

## Nv 12 — FileHelper
Реалізовано методи:
- GetExtension(string fileName) — повертає розширення файлу  
- IsImageFile(string fileName) — перевіряє, чи є файл зображенням  

Підтримувані формати зображень:
- `.jpg`
- `.jpeg`
- `.png`
- `.gif`
- `.bmp`

##  Юніт-тести

Тестовий проєкт містить **10+ тестів**, які перевіряють:

- правильне визначення розширення файлу  
- перевірку підтримуваних форматів  
- некоректні дані  
- граничні випадки (порожній рядок, файл без розширення)  
- перевірку винятків  

Використано:
- Fact
- Theory
- InlineData
